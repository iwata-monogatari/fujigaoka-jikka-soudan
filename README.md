# fujigaoka-jikka-soudan
富士ヶ丘サービス株式会社が運営する、実家じまい・空き家・相続不動産に関する相談専用サイトです。磐田市・袋井市・掛川市・森町・浜松市一部エリアを対象に、親の施設入居後の住まい、相続した実家、遠方で管理できない空き家、古家付き土地、農地や山林、境界・道路・残置物が絡む悩みを、介護と不動産の両面から整理します。売却を急がせず、家族の状況、親御様の気持ち、思い出、地域の記憶を大切にしながら、売る・貸す・解体する・残す・管理する選択肢を一緒に考える、伴走型の静的ミニサイトです。公式LINE、電話、問い合わせフォームにつなげ、相談未満の不安も受け止めます。日々更新するブログではなく、相談導線として機能する看板を目指します。介護事業の現場感と不動産実務の調査力を生かし、地域に根ざした安心の入口を作ります。
