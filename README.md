# ふじがおか 実家じまい・空き家相談室

富士ヶ丘サービス株式会社が運営する、相続した実家・空き家・親の家の整理に特化した
不動産売却「相談」サイト（静的ミニサイト）です。

> 親の家、相続した実家、空き家のこと。売る前に、まず一緒に整理しませんか。

売却査定を急がせるサイトではなく、**介護と不動産の両方の視点**から、
「売る・貸す・解体する・残す・管理する」を一緒に整理する相談窓口として設計しています。

## 対応エリア
磐田市・袋井市・掛川市・森町・浜松市一部

## ファイル構成

```text
fujigaoka-jikka-soudan/
├── index.html                      … トップページ（1ページ完結型）
├── cases/
│   └── important-disclosure.html   … ガイド：重要事項説明書のどこを見るべきか
├── assets/
│   ├── css/style.css               … デザイン（くすみブルー × 生成り × テラコッタ）
│   ├── js/main.js                  … ナビ開閉・FAQ・スムーススクロール
│   └── img/                        … 画像（下記の差し替えが必要）
└── README.md
```

## デザイン（配色）
緑系の磐田物語・公式サイトと差別化するため、本サイトは別系統の配色です。

| 役割 | 色 | コード |
|---|---|---|
| 背景（生成り） | クリーム | `#faf7f0` |
| 主役 | 藍鼠ブルー | `#44637a` / 濃 `#324c5e` |
| 差し色 | テラコッタ | `#c97b5a` |

色を変えたい場合は `assets/css/style.css` 冒頭の `:root` 内の変数を編集してください。

## 公開前に差し替える項目（プレースホルダー）
HTML内に以下のダミーが入っています。本番前に置き換えてください。

- 電話番号：`tel:0000000000` と表示文言（`000-000-0000`）
- LINEリンク：`href="#"`（LINE公式アカウントURLへ）
- 問い合わせフォームURL：`href="#"`
- 関連サイトリンク：富士ヶ丘サービス公式 / 不動産ページ / 磐田物語（`href="#"`）
- 会社情報：住所・宅建免許番号・受付時間・定休日
- 画像：`assets/img/hero.jpg`（実家・家族）、`oishi.jpg`（代表写真）など
  ※ 現在は画像なしでも成立するよう、プレースホルダー表示にしています。
- プライバシーポリシーページ

## 公開（Cloudflare Pages 想定）
1. GitHub リポジトリ `fujigaoka-jikka-soudan` にこの一式を push
2. Cloudflare Pages でリポジトリを接続
3. ビルド設定：フレームワークなし / 出力ディレクトリはルート（ビルド不要の静的サイト）
4. 公開後、スマホ表示と各リンクを確認
